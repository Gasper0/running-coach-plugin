# Strength Exercises for Runners

## Session Structure (30-45 min)
- Warm-up: 5 min dynamic mobility
- Main work: 3 rounds of the circuit below
- Cool-down: 5 min stretching

## Core Circuit (choose 4-5 exercises per session)

### Glutes & Hip Strength
| Exercise | Sets x Reps | Notes |
|----------|------------|-------|
| Hip Thrust (bodyweight or loaded) | 3x15 | Activate glutes at top |
| Single-leg Romanian Deadlift | 3x10 each | Keep back straight |
| Lateral Band Walk | 3x15 each | Mini-band around ankles |
| Clamshells | 3x20 each | Band optional |
| Glute Bridge | 3x20 | Squeeze at top for 2s |

### Core & Stability
| Exercise | Sets x Duration | Notes |
|----------|----------------|-------|
| Plank | 3x30-60s | Hips level, engage core |
| Side Plank | 3x20-30s each | Progress to hip dips |
| Dead Bug | 3x10 each side | Lower back flat on floor |
| Pallof Press | 3x12 each | Anti-rotation strength |
| Bird Dog | 3x10 each | Slow and controlled |

### Single-Leg Stability (injury prevention)
| Exercise | Sets x Reps | Notes |
|----------|------------|-------|
| Single-leg Squat (pistol progression) | 3x8 each | Use support if needed |
| Step-up | 3x12 each | Control the descent |
| Reverse Lunge | 3x12 each | Good knee tracking |
| Single-leg Calf Raise | 3x15 each | Off a step for full range |

### Calf & Achilles
| Exercise | Sets x Reps | Notes |
|----------|------------|-------|
| Calf Raise (double leg) | 3x20 | Full range of motion |
| Eccentric Calf Lower | 3x10 each | Slow 3-count descent |
| Jump Rope | 2x1 min | Light, quick contacts |

### Upper Body / Posture (running form)
| Exercise | Sets x Reps | Notes |
|----------|------------|-------|
| Push-up | 3x12 | Core tight |
| Dumbbell Row | 3x12 each | Keeps shoulders square |
| Shoulder Y-T-W | 2x10 | Lightweight, posture focus |

## Session Templates by Phase

### Base Phase (weeks 1-4)
Focus: build foundation, no DOMS before hard running sessions
- Hip Thrust, Glute Bridge, Plank, Dead Bug, Calf Raise, Step-up
- 2-3 sets, moderate reps, bodyweight or light load

### Build Phase (weeks 5-8)
Focus: single-leg power, functional strength
- Single-leg RDL, Pistol Squat progression, Side Plank, Pallof Press, Eccentric Calf
- 3 sets, heavier if possible, focus on control

### Peak Phase (weeks 9-10)
Focus: maintain, no fatigue — short sessions only (20 min)
- Glute Bridge, Clamshells, Bird Dog, Plank
- No heavy loading, no new exercises

### Taper
Skip strength sessions the week before the race. At most 1 light session of mobility/activation only.

## Post-Run Stretching (15-20 min)

Hold each stretch 30-45 seconds, both sides:
1. Hip flexor lunge stretch
2. Pigeon pose (glute/piriformis)
3. Standing quad stretch
4. Seated hamstring stretch
5. Calf stretch (straight leg + bent knee for soleus)
6. IT band / cross-body stretch
7. Thoracic rotation seated

## Foam Rolling (5-10 min)
- Quads
- IT band
- Calves
- Glutes / piriformis
- Upper back
