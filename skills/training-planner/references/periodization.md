# Periodization Rules

> Phase structure, weekly volume progression, and stepback week placement.
> The skill follows these rules unless the user explicitly overrides them.

---

## Phase definitions

Every plan is divided into 4 phases. Each has a distinct purpose and intensity profile.

### Base — Foundation building
- **Goal**: aerobic base, mileage tolerance, injury prevention
- **Quality sessions**: 0-1 per week, mostly strides and progression runs
- **Volume**: low to moderate, building progressively
- **Long run**: gradually extended, all easy pace
- **Strength**: 2x/week, full body
- **Hyrox add-on**: 1 light Run Brick if Hyrox prep is concurrent

### Build — Specific endurance
- **Goal**: improve threshold and tempo capacity
- **Quality sessions**: 2 per week (typically Tue + Thu)
- **Volume**: at or near peak weekly mileage
- **Long run**: peak distance, sometimes with a fast finish
- **Strength**: 2x/week, slight reduction in volume
- **Hyrox add-on**: 1 Run Brick at moderate intensity

### Peak — Race-specific sharpening
- **Goal**: hone race-specific systems (VMA, race pace)
- **Quality sessions**: 2 per week, sharper and more specific
- **Volume**: slight drop from Build peak (90%)
- **Long run**: race-specific (marathon-pace segments for marathon prep, etc.)
- **Strength**: 1-2x/week, maintenance only
- **Hyrox add-on**: 1 Run Brick at race intensity

### Taper — Recovery and readiness
- **Goal**: preserve fitness, drop fatigue, sharpen
- **Quality sessions**: 1 per week, very short and sharp
- **Volume**: 50-70% of Peak by mid-taper, ~40% in final week
- **Long run**: cut by 30-50%
- **Strength**: light maintenance only, none in final week
- **Final 3 days**: very easy, minimal volume

## Phase durations by distance

These are defaults — adjust based on user availability and starting fitness.

### 10km (6-10 weeks total)

| Phase | Weeks | Focus |
|---|---|---|
| Base | 2-3 | Aerobic base, 1 strides session per week |
| Build | 2-3 | Threshold + tempo, 30-40 min sessions |
| Peak | 2 | VMA + 10k-pace work |
| Taper | 1 | Cut volume 50%, 1 short quality |

### Half-marathon (10-12 weeks total)

| Phase | Weeks | Focus |
|---|---|---|
| Base | 3 | Aerobic base, gradual mileage build |
| Build | 3-4 | Threshold dominant, 2x/week quality |
| Peak | 2-3 | Race-pace and short threshold |
| Taper | 1-2 | Cut volume 40-50%, race-pace strides |

### Marathon (14-18 weeks total)

| Phase | Weeks | Focus |
|---|---|---|
| Base | 4-5 | Build mileage to 80% of peak |
| Build | 5-6 | Threshold + long run development |
| Peak | 3 | Marathon-pace work, peak long runs |
| Taper | 2-3 | Significant volume cut, intensity preserved short |

### Hyrox (8-12 weeks total)

| Phase | Weeks | Focus |
|---|---|---|
| Base | 2-3 | Aerobic base + station familiarity |
| Build | 3-4 | Run Brick at moderate intensity, tempo |
| Peak | 2-3 | Run Brick at race intensity, compromised run |
| Taper | 1 | Cut volume, 1 light Brick early week |

## The 10% rule

**Never increase weekly volume by more than 10% from one week to the next.**

This is a hard ceiling, not a target. Going from 30km to 40km in one week is a 33% jump — almost guaranteed injury risk for the average runner.

### How to apply

For a 10km plan starting from a current weekly volume of 35km:

| Week | Volume | Increase |
|---|---|---|
| 1 | 35 | baseline |
| 2 | 38 | +9% |
| 3 | 41 | +8% |
| 4 (stepback) | 31 | -25% |
| 5 | 41 | back to W3 |
| 6 | 45 | +10% |
| 7 | 49 | +9% |
| 8 (stepback) | 37 | -25% |
| 9 (taper) | 30 | -19% |
| 10 (race week) | 18 | -40% |

The plan is a sawtooth pattern: build → build → build → stepback, repeat.

## Stepback weeks

A stepback week is a week of reduced volume to allow recovery and adaptation. Required in any plan longer than 8 weeks.

### Placement rules

- Every 3-4 weeks of progressive build
- Always 25-30% volume reduction vs the prior week
- Quality sessions reduced in volume but not removed entirely (1 short quality is fine)
- Long run cut by 30-40%

### When NOT to do a stepback

- The week before a stepback is already low because of life events (travel, illness, missed sessions) — skip the stepback, continue progression instead
- In the final 2 weeks before a marathon (taper handles recovery)

## Weekly structure templates

### 5 sessions/week (most common)

| Day | Session | Notes |
|---|---|---|
| Mon | Rest | Full recovery |
| Tue | Quality (VMA or threshold) | Fresh legs |
| Wed | Easy run | Recovery run, Z1-Z2 |
| Thu | Quality (tempo or threshold) | Mid-week sharpening |
| Fri | Easy run or rest | Pre-weekend recovery |
| Sat | Strength (legs) + short easy | Sets up Sunday back-to-back |
| Sun | Long run | Volume peak of the week |

### 4 sessions/week (less time)

| Day | Session |
|---|---|
| Tue | Quality (VMA or threshold) |
| Thu | Tempo or threshold |
| Sat | Easy run or strength |
| Sun | Long run |

### 6 sessions/week (advanced)

| Day | Session |
|---|---|
| Mon | Rest |
| Tue | Quality (VMA) |
| Wed | Easy run + strides |
| Thu | Quality (threshold) |
| Fri | Easy recovery run |
| Sat | Strength legs + short easy |
| Sun | Long run |

## Hyrox-specific weekly structure

If the user is preparing Hyrox concurrently:

| Day | Session |
|---|---|
| Mon | Rest or mobility |
| Tue | VMA (running) |
| Wed | Easy run |
| Thu | Threshold or tempo |
| **Fri** | **Hyrox Run Brick** (running + 1-2 stations) |
| Sat | Strength legs |
| Sun | Long run |

The Run Brick replaces what would otherwise be a third quality day. Friday is chosen because it's right before strength legs (Sat) and long run (Sun) — the Hyrox stimulus is "fresh enough" without competing with the harder days.

---

## Adjusting to user availability

If the user has less than 4 days available, propose a 3-session structure:
- Tue: Quality (alternating VMA / threshold week to week)
- Thu: Tempo or moderate intensity
- Sun: Long run

3 sessions can produce real progress for a 10km goal, less so for a marathon. Flag this if the goal is ambitious.

If the user has irregular availability (e.g., shift work), build the plan around their fixed days, and prioritize quality session protection over weekly structure consistency.

## Volume defaults by distance (peak week)

These are typical peak-week volumes for amateur runners. The user's actual peak depends on starting fitness and time available.

| Distance | Peak weekly volume | Peak long run |
|---|---|---|
| 10km | 40-55 km | 16-20 km |
| Half-marathon | 50-70 km | 20-25 km |
| Marathon | 65-100 km | 30-35 km |
| Hyrox | 35-50 km (+ stations) | 16-20 km |

These are not goals to chase — they're typical numbers. A runner doing 40 km/week can run a sub-1h30 half-marathon with smart training.
